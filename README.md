# multi-websites-searching-tool：主要網站商品搜尋器
一個簡單練習Flask的小工具，選擇目前主要的購物網站一鍵搜尋，免除搜尋引擎搜尋後還須逐一確認查找的困擾。並使用GCP Compute Engine服務以容器化方式部署網站。
